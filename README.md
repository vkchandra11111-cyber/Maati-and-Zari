# Maati and Zari — Website

Handpicked Indian ethnic wear for the diaspora. Sarees, lehengas, sherwanis, and kurtas shipped worldwide. DM to shop.

## What's here

- `index.html` — the complete website. No build step, no dependencies. All styles are inline; fonts load from Google Fonts.

## Deploy on GitHub Pages (free)

1. Create a new repository on GitHub (e.g. `maati-and-zari`).
2. Upload `index.html` (and this README) to the repository.
3. Go to **Settings → Pages**.
4. Under **Source**, choose **Deploy from a branch**, select `main` and `/ (root)`, then **Save**.
5. Your site will be live in a minute or two at `https://<your-username>.github.io/maati-and-zari/`.

## Before going live

- Replace every `https://instagram.com` link with your actual Instagram profile URL (search the file for `instagram.com` — replace every one of them).
- Update the footer year/text if needed.
- Optional: replace the CSS fabric swatches in the Collections section with real product photos by swapping each `.swatch` div for an `<img>`.
